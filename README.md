Aplicación para gestionar conciertos --> Conectar la aplicación con una base de datos (SQL).
* Métodos para agregar, eliminar, y listar.
* Clases: Entrada, Concierto y Artista.
* Opción para generar un resumen de los datos introducidos de Artista y Concierto.
* Opción para generar el recibo de la entrada comprada.

Como ejecutar:
* En el menu, se escoge una opción del 0 al 10 según lo que se desee hacer.
* Los datos generados serán guardados en las tablas de la base de datos.
